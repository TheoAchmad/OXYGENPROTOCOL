# 🚀 OXYGEN PROTOCOL
**Unity-based Survival Game with Coding Challenges**

Oxygen Protocol adalah game survival 3D/2D (sesuaikan) yang dibuat menggunakan **Unity Engine**. Hal yang unik dari game ini adalah mekanik bertahannya yang mengharuskan pemain menyelesaikan tantangan logika pemrograman untuk bertahan hidup.

## 🛠 Tech Stack
- **Engine:** Unity 6.3 LTS (atau versi yang kamu pakai)
- **Language:** C#
- **Platform:** PC / WebGL

## ✨ Fitur Utama
- **Coding Terminal:** Pemain harus mengetikkan perintah/logika untuk membuka pintu atau memperbaiki sistem oksigen.
- **Survival System:** Manajemen sumber daya oksigen yang sinkron dengan progres pemain.
- **Environment Interaction:** Interaksi objek menggunakan sistem Raycast/Trigger di Unity.

## 🏗 Cara Menjalankan
1. Clone repositori ini.
2. Buka folder proyek menggunakan **Unity Hub**.
3. Pastikan menggunakan versi Unity yang sesuai.
4. Tekan **Play** pada scene `MainScene`.
